File Relationships:
-------------------

SpellRegistry (Autoload)
    └─> Provides spell data to all spell classes

Player
    ├─> ManaManager
    │       └─> Manages mana pool, regeneration, consumption
    └─> SpellManager
            ├─> Creates spell instances
            └─> Manages equipped spells
                    └─> SpellBase instances
                            ├─> ProjectileSpell
                            │       └─> Spawns SpellProjectile nodes
                            ├─> BeamSpell
                            │       └─> Creates beam visuals
                            └─> AOESpell
                                    └─> Creates AOE effects

Getting Started:
----------------

1. Register SpellRegistry as autoload in Project Settings
2. Add ManaManager and SpellManager nodes to player
3. Set up input actions (cast_spell, next_spell, etc.)
4. Create basic UI (mana bar, spell hotbar)
5. Implement enemy damage/status methods
6. Test with: spell_manager.equip_spell(0, "fireball")

See QUICK_START.md for detailed setup instructions.
